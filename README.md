# World On Time

Premium Next.js rebuild of World On Time for Vercel.

## What is included

- Responsive Apple-like homepage design
- Live world clock
- Popular city clocks
- Time zone converter
- Countdown timer
- Stopwatch
- Pomodoro timer
- SEO metadata
- Sitemap and robots files
- `ads.txt` for Google AdSense readiness
- Privacy Policy, Terms of Service, and FAQ pages

## Run locally

Install dependencies:

```bash
npm install
```

Start the local preview:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Publish with GitHub and Vercel

1. Create a new GitHub repository named `world-on-time`.
2. Upload or push this project folder to that repository.
3. Go to Vercel and choose **Add New Project**.
4. Import the GitHub repository.
5. Keep the default Next.js settings.
6. Click **Deploy**.
7. After the preview looks correct, connect `worldontime.info` in Vercel.

## Before replacing the Durable website

Keep Durable live until the Vercel deployment is tested. Only move the domain to Vercel after the preview deployment is approved on desktop and mobile.
